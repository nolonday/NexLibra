# NexLibra

## 📚 Название проекта

NexLibra — веб-сайт цифровой библиотеки

---

## 🎯 Цель проекта

Целью проекта является разработка веб-приложения цифровой библиотеки, предоставляющего пользователям удобный доступ к каталогу книг, информации об авторах, новинкам литературы, а также контактным данным библиотеки.

Проект направлен на автоматизацию процессов поиска и просмотра учебной и художественной литературы, а также повышение удобства взаимодействия пользователей с библиотечным фондом.

---

## 🛠️ Стек технологий

### Frontend:

- React (Vite)
- JavaScript (ES6+)
- HTML5
- CSS3
- React Router (для маршрутизации)
- Axios (для работы с API)

### Backend:

- Node.js
- Express.js
- REST API
- JWT (аутентификация)
- bcrypt (хеширование паролей)
- CORS
- dotenv

### Database:

- PostgreSQL
